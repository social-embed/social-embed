## Inner `<iframe>`

Optional, [Custom CSS Properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties) were added in `@social-embed/wc` at `0.0.3-next.0`:

- `--social-embed-iframe-height`
- `--social-embed-iframe-width`

### Height
#### HTML

```html
<style>
o-embed.full {
  --social-embed-iframe-height: 100%;
  --social-embed-iframe-width: 100%;
  height: 500px;
}
</style>
<o-embed url="https://youtu.be/FTQbiNvZqaY" class="full"></o-embed>
```
<o-embed url="https://youtu.be/FTQbiNvZqaY" style={{height:"500px"}} class="full"></o-embed>

### Height + width

```html
<style>
o-embed.full {
  --social-embed-iframe-height: 100%;
  --social-embed-iframe-width: 100%;
  height: 300px;
  width: 500px;
}
</style>
<o-embed url="https://youtu.be/FTQbiNvZqaY" class="full"></o-embed>
```
<o-embed url="https://youtu.be/FTQbiNvZqaY" style={{height:'300px',width:'500px'}} class="full"></o-embed>
