`@social-embed` is two tools:

1. [lib](./lib/index.mdx): A library of javascript / TypeScript functions and regexes to grab URLs
2. [wc](./wc/index.mdx): A [web-component](https://developer.mozilla.org/en-US/docs/Web/Web_Components) to switch out embed tags
