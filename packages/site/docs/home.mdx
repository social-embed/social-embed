For a view from 30,000 feet, `@social-embed` is comprised of two packages:

| Name    | Package               | Source location  | Type            | Description                                                                                  |
| ------- | --------------------- | ---------------- | --------------- | -------------------------------------------------------------------------------------------- |
| [`lib`] | [`@social-embed/lib`] | [/packages/lib/] | Library         | Repository of regexes / parsers for embed-friendly websites (e.g. YouTube)                   |
| [`wc`]  | [`@social-embed/lib`] | [/packages/wc/]  | [Web component] | Renders native web players via `<o-embed>`. Built with [LitElement] + [TypeScript] + [`lib`] |
