## Quick Start

### Installing via Package Managers

<Tabs syncKey="pkg">
<TabItem label="npm">

```bash
npm i @social-embed/lib
```

</TabItem>

<TabItem label="yarn">

```bash
yarn add @social-embed/lib
```

</TabItem>

<TabItem label="pnpm">

```bash
pnpm add @social-embed/lib
```

</TabItem>
</Tabs>

### Using a CDN

For environments that support ES modules from remote sources (e.g., online sandbox environments), you can import the library directly:

<Tabs syncKey="cdn">
<TabItem label="esm.sh (recommended)">

```typescript
import { MatcherRegistry } from "https://esm.sh/@social-embed/lib";

const registry = MatcherRegistry.withDefaults();
const embedUrl = registry.toEmbedUrl("https://youtu.be/abc123");
```

</TabItem>
<TabItem label="unpkg">

```typescript
import { MatcherRegistry } from "https://www.unpkg.com/@social-embed/lib?module";
```

</TabItem>
<TabItem label="jsdelivr">

```typescript
import { MatcherRegistry } from "https://cdn.jsdelivr.net/npm/@social-embed/lib/+esm";
```

</TabItem>
</Tabs>
