## Quickstart

### Packages

<Tabs syncKey="pkg">
<TabItem label="npm">

```bash
npm i @social-embed/lib
```

</TabItem>

<TabItem label="yarn">

```bash
yarn add @social-embed/lib
```

</TabItem>

<TabItem label="pnpm">

```bash
pnpm add @social-embed/lib
```

</TabItem>
</Tabs>

### via CDNs

If you can support ESM via remote sources URLs (such as via code sandboxes or late [snowpack]'s [streaming imports]), you can also import like this:

<Tabs syncKey="cdn">
<TabItem label="unpkg">

```typescript
import { getYouTubeIdFromUrl } from 'https://www.unpkg.com/@social-embed/lib?module';
```

</TabItem>
<TabItem label="skypack">

```typescript
import { getYouTubeIdFromUrl } from 'https://cdn.skypack.dev/@social-embed/lib';
```

</TabItem>
</Tabs>

[streaming imports]: https://www.snowpack.dev/guides/streaming-imports
[snowpack]: https://www.snowpack.dev
