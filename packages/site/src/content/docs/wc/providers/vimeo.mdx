## Vimeo-specific Options

The `<o-embed>` component supports Vimeo-specific options via the `provider-options` attribute.

### Available Options

| Option | Type | Description |
|--------|------|-------------|
| `autoplay` | boolean | Auto-start the video (requires `muted: true` in most browsers) |
| `muted` | boolean | Start the video muted |
| `loop` | boolean | Loop the video continuously |
| `background` | boolean | Enable background mode (hides all UI, paid accounts only) |
| `autopause` | boolean | Pause when video scrolls out of view (default: true) |
| `controls` | boolean | Show player controls (default: true) |
| `title` | boolean | Show video title (default: true) |
| `byline` | boolean | Show author byline (default: true) |
| `portrait` | boolean | Show author portrait (default: true) |

### Autoplay with Muted

Enable autoplay (requires muted for most browsers):

```html
<o-embed
  url="https://vimeo.com/134668506"
  provider-options='{"autoplay": true, "muted": true}'
></o-embed>
```

### Loop

Loop the video continuously:

```html
<o-embed
  url="https://vimeo.com/134668506"
  provider-options='{"loop": true}'
></o-embed>
```

### Background Mode

<Aside type="caution">
Background mode is only available for paid Vimeo accounts. It hides all player UI elements.
</Aside>

```html
<o-embed
  url="https://vimeo.com/134668506"
  provider-options='{"background": true}'
></o-embed>
```

### Hide UI Elements

Hide specific UI elements for a cleaner appearance:

```html
<o-embed
  url="https://vimeo.com/134668506"
  provider-options='{"title": false, "byline": false, "portrait": false}'
></o-embed>
```

### Disable Autopause

Prevent the video from pausing when scrolled out of view:

```html
<o-embed
  url="https://vimeo.com/134668506"
  provider-options='{"autopause": false}'
></o-embed>
```

### Hide Controls

```html
<o-embed
  url="https://vimeo.com/134668506"
  provider-options='{"controls": false}'
></o-embed>
```

### Combining Options

```html
<o-embed
  url="https://vimeo.com/134668506"
  provider-options='{"autoplay": true, "muted": true, "loop": true, "title": false}'
></o-embed>
```
