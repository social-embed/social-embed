## Supported URL formats

| Format | Example |
|--------|---------|
| Standard | `https://vimeo.com/134668506` |
| With www | `https://www.vimeo.com/134668506` |
| Channel | `https://vimeo.com/channels/staffpicks/134668506` |
| Group | `https://vimeo.com/groups/shortfilms/videos/134668506` |
| Video path | `https://vimeo.com/video/134668506` |
| Player subdomain | `https://player.vimeo.com/video/134668506` |
