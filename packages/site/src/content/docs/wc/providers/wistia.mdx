## Wistia-specific Options

The `<o-embed>` component supports Wistia-specific options via the `provider-options` attribute.

### Available Options

| Option | Type | Description |
|--------|------|-------------|
| `autoPlay` | boolean | Auto-start the video (note: camelCase) |
| `muted` | boolean | Start the video muted (required for autoplay in most browsers) |
| `playerColor` | string | Custom player accent color (hex without #) |
| `videoFoam` | boolean | Enable responsive sizing |

<Aside type="note">
Wistia uses camelCase for its parameters (e.g., `autoPlay` not `autoplay`).
</Aside>

### Autoplay with Muted

Enable autoplay (requires muted for most browsers):

```html
<o-embed
  url="https://support.wistia.com/medias/26sk4lmiix"
  provider-options='{"autoPlay": true, "muted": true}'
></o-embed>
```

### Player Color

Customize the player accent color using a hex code (without the # prefix):

```html
<o-embed
  url="https://support.wistia.com/medias/26sk4lmiix"
  provider-options='{"playerColor": "ff69b4"}'
></o-embed>
```

### Video Foam (Responsive)

Enable responsive sizing that adapts to the container:

```html
<o-embed
  url="https://support.wistia.com/medias/26sk4lmiix"
  provider-options='{"videoFoam": true}'
></o-embed>
```

### Combining Options

```html
<o-embed
  url="https://support.wistia.com/medias/26sk4lmiix"
  provider-options='{"autoPlay": true, "muted": true, "playerColor": "ff69b4"}'
></o-embed>
```
