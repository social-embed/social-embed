[Wistia] is a marketing-oriented video platform.

[wistia]: https://wistia.com

## Supported URL formats

| Format | Example |
|--------|---------|
| Support medias | `https://support.wistia.com/medias/26sk4lmiix` |
| Support embed | `https://support.wistia.com/embed/26sk4lmiix` |
| Fast subdomain | `https://fast.wistia.com/medias/26sk4lmiix` |
| Short domain | `https://support.wi.st/medias/26sk4lmiix` |

## Default dimensions

URL: https://support.wistia.com/medias/26sk4lmiix

```html
<o-embed url="https://support.wistia.com/medias/26sk4lmiix"></o-embed>
```

<o-embed
