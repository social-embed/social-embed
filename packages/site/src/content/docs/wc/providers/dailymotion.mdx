## DailyMotion-specific Options

The `<o-embed>` component supports DailyMotion-specific options via the `provider-options` attribute.

### Available Options

| Option | Type | Description |
|--------|------|-------------|
| `mute` | boolean | Start the video muted |
| `startTime` | number | Start time in seconds |
| `loop` | boolean | Loop the video continuously |
| `playerId` | string | Custom Player ID (for advanced customization) |

### Muted Playback

```html
<o-embed
  url="https://www.dailymotion.com/video/x7znrd0"
  provider-options='{"mute": true}'
></o-embed>
```

### Start Time

Start playback at a specific time (in seconds):

```html
<o-embed
  url="https://www.dailymotion.com/video/x7znrd0"
  provider-options='{"startTime": 30}'
></o-embed>
```

### Loop

Enable continuous looping:

```html
<o-embed
  url="https://www.dailymotion.com/video/x7znrd0"
  provider-options='{"loop": true}'
></o-embed>
```

### Combining Options

```html
<o-embed
  url="https://www.dailymotion.com/video/x7znrd0"
  provider-options='{"mute": true, "startTime": 30, "loop": true}'
></o-embed>
```

## Custom Player ID

For advanced customization, you can use a Player ID created in the DailyMotion Partner HQ:

```html
<o-embed
  url="https://www.dailymotion.com/video/x7znrd0"
  provider-options='{"playerId": "xc394"}'
></o-embed>
```

<Aside type="tip">
Player IDs allow you to customize player appearance and behavior beyond the basic options. Create them in the [DailyMotion Partner HQ](https://www.dailymotion.com/partner/).
</Aside>

## CSS Custom Properties

Use CSS custom properties for responsive sizing:

```html
<style>
  o-embed.full {
    --social-embed-iframe-height: 100%;
    --social-embed-iframe-width: 100%;
    width: 100%;
    height: 450px;
  }
</style>
<o-embed url="https://www.dailymotion.com/video/x7znrd0" class="full"></o-embed>
```

<o-embed url="https://www.dailymotion.com/video/x7znrd0" style={{height:'450px',width:'100%'}} class="full"></o-embed>
