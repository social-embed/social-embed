:::caution[Autoplay issues]
As of March 2025, autoplay support via iframes is broken for DailyMotion videos. For more information, see ["How to work with autostart"](https://developers.dailymotion.com/guides/player-autostart/) in the DailyMotion developer documentation.
:::

## Custom dimensions

```html
<o-embed
  url="https://www.dailymotion.com/video/x7znrd0"
  width="1080"
  height="536"
></o-embed>
```

<o-embed url="https://www.dailymotion.com/video/x7znrd0" width="1080" height="536"></o-embed>

## Custom dimensions (more)

```html
<style>
  o-embed.full {
    --social-embed-iframe-height: 100%;
    --social-embed-iframe-width: 100%;
    width: 100%;
    height: 450px;
  }
</style>
<o-embed url="https://www.dailymotion.com/video/x7znrd0" class="full"></o-embed>
```

<o-embed url="https://www.dailymotion.com/video/x7znrd0" style={{height:'450px',width:'100%'}} class="full"></o-embed>
