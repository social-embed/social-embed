# CMS Content Example

This example shows two content-storage paths: stored HTML that already contains `<o-embed>` and a structured URL field rendered into the same tag shape.

Install dependencies:

```bash
pnpm install
```

Start the dev server:

```bash
pnpm dev
```

Run all tests:

```bash
pnpm test
```

Run the full local verification suite:

```bash
pnpm verify
```

Run browser tests:

```bash
pnpm test:browser
```

Run unit tests only:

```bash
pnpm test:unit
```

Repo path:

```bash
examples/integration-patterns/cms-content
```

StackBlitz GitHub-subdir URL pattern:

```text
https://stackblitz.com/github/social-embed/social-embed/tree/BRANCH/examples/integration-patterns/cms-content
```
